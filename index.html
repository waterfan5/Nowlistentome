Waterfan5's Toolbox for Mystery Geocaches

Text Tools

  • Anagram Helper - Analyze which words can be made from a set of letters
  • Expand Text - Expand text based and letter and number values
  • Highlight Text - Find and highlight words in text
  • Pattern Detector - Looks for repeating patterns in strings
  • Text Analyzer - Calculates occurences of letters in strings
  • Transform Text - Manipulate text in many different ways

Decoding Tools

  • Base Converter - Convert numbers from and to different base systems. 
  • Binary Attack - Decode two-character numbers systm like binary, braille, morse etc 
  • HTML Text Extractor - Extract text from HTML based on Bacon cipher principles 

Cipher Tools

  • Book Cipher - Process book ciphers 
  • Rotate and Bash - Performs mass basic operations like of caesar/ROT-x, atbash and reversing on text
  • Substitution Test - Quick experimentation of single or multiple letters plaintext substitution ciphers
  • Sentence Mapper - Encodes and decodes a sentence using a key string with non-unique letters
  • Quagmire Crib - Vigenere and Quagmire analyses with known plain-text

Map Tools

  • Geo Calculator - For map based calculations 
  • Coordinate Generator - Generate full coordinates based on partial information

Visual Tools

  • Character Mapper - Easy way to map characters on an image to a letter or number 
  • Digital Clock - Visualizes different formats as a digital clock
  • Grid Paint - Paints using grid coordinates
  • Image Mapper - Maps the colors in an image to letters
  • Image Recolor - Recolors an image using some standard filters

Special Tools

  • Geocaching Solving Checklist - Simple checklist for looking for hints on a cache page
  • Logic Helper - Analyzes logic clues and solves logic grid puzzles
  • Quotefall Solver - Attempts to solve quotefall puzzles
  • Strands Solver - Attempts to solve Strands puzzles
  • Solve ABC - Solve simple cryptarithm equations 
  • Quick lookup - Simple way to lookup presidents, states, chemical elements,etc by code

Tampermonkey add-ins

  • Puzzle Helper - Useful additions to a cache page for puzzle solvers
  • Solution Hider - Hides the solution, notes etc from Geocaching page
  • Jigidi Solver - Recolors the jigidi and automatically matches pieces based on color.
